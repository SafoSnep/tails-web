Tails is built to provide a great way to browse interesting content on your mobile devices with a carefully crafted, fully native, touchscreen focused UI fitting right into modern iOS. It currently supports the following features:

  • Searching and Tag Autocompletion
  • Full-res File Downloads
  • Animated GIFs and PNGs
  • Blacklisting (if logged in)
  • Infinite Scrolling
  • Hi-Res Images
  • iPhone and iPad Support

Tails also enhances browsing with features that are currently not possible on any iOS web browser:

  • Light and Dark Themes
  • Local Tag Subscriptions
  • WebM Compatibility (with 3rd party apps)

Download and Install

Please don't go looking for Tails in the App Store – its rules deny an app like Tails point-blank. Instead, the suggested installation method is to “sideload” the app via AltStore. No jailbreak required!

Quick Start

  1. (Install AltStore)
  2. Open this page with Safari on the same device you've installed AltStore
  3. Tap the “Download .ipa” button above and the download will begin in the top right corner
  4. Tap the downloaded file after it's finished downloading
  5. Tap the Share Icon in the top right corner

Detail View

iPadOS Enhancements

Tails supports iPadOS's Drag-and-Drop system with multiple items, and multitasking with Split Screen and Slide Over.

It works great with 3rd party apps, such as VizRef, and for saving multiple images to your device or any cloud service in your Files app by dragging them over.

(Alternatively, save single images to your device by long pressing on them, for example on devices without drag-and-drop support.)

Expanded Layout

On iPad, a lot more thumbnails are shown per screen, with even more thumbnails available as an optional layout enabled in the app's settings.
